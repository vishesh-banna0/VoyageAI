VoyageAI is an intelligent trip planner agent that generates complete, structured travel plans for any city worldwide using real-time data. It combines live information with reasoning-based planning to create optimized, budget-aware itineraries.

---

## Features

- **Real-Time Weather Info**  
  Retrieves live weather forecasts to influence activity planning and scheduling.

- **Attractions & Activities**  
  Discovers top landmarks, cultural spots, events, and recommended experiences.

- **Hotel Cost Estimation**  
  Suggests accommodation options with estimated pricing based on stay duration and category.

- **Currency Conversion**  
  Converts total trip costs into the user's preferred currency using real-time exchange rates.

- **Intelligent Itinerary Planning**  
  Generates structured day-by-day plans with optimized activity sequencing.

- **Total Expense Calculation**  
  Aggregates hotel, activities, food, and miscellaneous expenses into a clear budget breakdown.

- **Overall Trip Summary**  
  Provides a concise travel overview including highlights, costs, and key recommendations.

---

## How It Works

VoyageAI uses an agentic AI architecture that:

1. Understands user goals (destination, budget, duration, preferences)  
2. Fetches real-time data via integrated tools  
3. Reasons across options to optimize plans  
4. Produces a structured itinerary and cost summary
